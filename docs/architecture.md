# TravAgent - 智能旅行助手架构说明

## 1. 项目概述

TravAgent 是一个基于大语言模型的智能旅行规划助手，采用多智能体协作架构，能够根据用户的预算、时间和偏好生成个性化的旅行方案。

### 2.2 智能体职责

| 智能体 | 职责 | 核心能力 |
|--------|------|----------|
| TravelManager | 整体协调与调度 | 任务分解、智能体调度、冲突检测 |

| ContentAgent | 内容生成 | 景点介绍、行程描述、文案美化 |
| FoodAgent | 美食推荐 | 餐厅推荐、特色美食、避免重复 |

## 3. 状态机设计

### 3.1 行程规划状态流转

```
start → input_info → plan_generation → review → [confirm | modify] → output
     ↑                                                      │
     └───────────────────────────────────────────────────────┘
```

### 3.2 状态节点说明

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| start | 初始状态 | 用户开始对话 |
| input_info | 收集信息 | 等待用户输入目的地、天数、预算 |
| plan_generation | 生成方案 | 信息收集完成 |
| review | 方案审核 | 方案生成完成 |
| confirm | 确认方案 | 用户确认 |
| modify | 修改方案 | 用户不满意，需要调整 |
| output | 输出结果 | 方案最终确认 |

## 4. 数据流设计

### 4.1 核心数据模型

```python
# 行程计划
class Itinerary:
    destination: str       # 目的地
    days: int              # 天数
    budget: float          # 预算
    start_date: datetime   # 出发日期
    schedule: List[Day]    # 每日行程
    
# 每日行程
class Day:
    day_number: int              # 第几天
    date: datetime              # 日期
    morning: Activity           # 上午活动
    lunch: FoodRecommendation   # 午餐推荐
    afternoon: Activity         # 下午活动
    dinner: FoodRecommendation  # 晚餐推荐
    hotel: HotelRecommendation  # 酒店推荐
    
# 活动
class Activity:
    name: str            # 景点名称
    type: str            # 类型：文化古迹/自然景观/娱乐
    rating: float        # 评分
    price: float         # 门票价格
    duration: int        # 预计时长(分钟)
    description: str     # 介绍
    
# 美食推荐
class FoodRecommendation:
    name: str            # 餐厅名称
    type: str            # 菜系
    rating: float        # 评分
    price_level: str     # 价格等级
    address: str         # 地址
```

### 4.2 数据流图

```

              │              │              │              │
              ↓              ↓              ↓              ↓
         参数校验       API调用        冲突检测        文案美化
```

## 5. 工具定义

### 5.1 工具列表

| 工具名称 | 功能 | API来源 |
|----------|------|----------|
| search_attractions | 搜索景点 | 高德地图POI搜索 |
| search_hotels | 搜索酒店 | 高德地图POI搜索 |
| search_restaurants | 搜索餐厅 | 高德地图POI搜索 |
| get_transit_route | 获取交通路线 | 高德地图路径规划 |
| get_weather | 获取天气 | 和风天气API |
| calculate_cost | 费用计算 | 本地计算 |
| generate_map | 生成地图 | 高德地图+HTML渲染 |

### 5.2 工具调用流程

```
Agent → ToolCallRequest → ToolExecutor → API Response → ToolResult → Agent
```

## 8. 扩展性设计

### 8.1 可扩展模块

- **智能体扩展**: 可添加购物助手、摄影助手等新智能体
